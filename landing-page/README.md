# CivicPulse Landing Page

This is a standalone version of the CivicPulse landing page, designed for showcase and deployment on Vercel.

## Features
- Full visual fidelity of the original landing page
- All login and administrative navigation removed for public viewing
- Interactive modals (Privacy, Terms, Help Center)
- Responsive design with Tailwind CSS 4
- Smooth scroll navigation

## Development
```bash
npm install
npm run dev
```

## Deployment
This project is ready to be deployed to Vercel. Simply connect this directory as a new project in Vercel.
